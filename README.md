Parallax Website Clone

Project Description
This project is a **Parallax Scrolling Website** created as part of a Web Development assignment.  
The website is a single-page application that demonstrates advanced CSS styling, fixed navigation, and parallax background effects.  

The theme used for this project is a **Personal Portfolio**, showcasing information about the student, projects, and contact details.

 Features
- Semantic HTML structure (header, sections, footer)
- Fixed navigation bar with anchor links
- Three parallax background sections using CSS
- Google Maps iframe in the contact section
- Responsive design using media queries
- Hover effects using CSS pseudo-classes
- External Google Fonts integration

Technologies Used
- HTML5  
- CSS3  
- Google Fonts  
- Google Maps Embed API  
- Git & GitHub
